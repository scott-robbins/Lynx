How Do I Login to the page?

To log in, you have to be a registered node in the Lynx Peer to Peer Network. To do that you can visit the link below and download the source code. The README in the github repository has detailed instructions on installation.

What is Lynx

Lynx is a project that utilizes what is called "peer to peer" technology to make it possible to share files, send messages and distribute/retrieve resources in an organic network of users. Many popular companies use this technology, though it is often put in more abstract terms like "the cloud". Really what is going on here is an attempt to leverage the growing inter-connectivity (internet) and memory of available machines to make end users benefit from their collective power (be it storage, routing, indexing, etc.) to do pretty crazy things. Think BitTorrent, Skype, TOR...
